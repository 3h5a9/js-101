Notes

Variable

let and var Value can be reasign.

let age = 30;
age = 31;
console.log(age);

const Value can not be reasign

const birthYear = 30;
birthYear = 31;
console.log(birthYear);

This is going to show you an error in the console.

Uncaught TypeError: invalid assignment to const 'birthYear'
